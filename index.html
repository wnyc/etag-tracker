etag-session

A simple cross platform session_id tracker that uses etags instead of cookies. Install your own server or feel free to use ours: 

   
    <script type="text/javascript">
      $( document ).ready (
        function() {
          $.getJSON("http://sessions.wnyc.net/session_id", 
                    function( data) { 
                      alert("You are session #" +  data["id"]);
                    }
          );
        }
      )

    </script>

Installation

If you want to run your own etag-tracker server, simply download and install

git clone git@github.com:wnyc/etag-tracker.git
cd etag-tracker/server
make

How does it work?

Its pretty dumb actually. When you make a call to the etag-tracker software it looks for a If-None-Match header. If there isn't one, it returns an etag value and a session id. If there is, it returns a 304 telling the browser to use whatever session_id it previously retrieved.
